# 17장 JavaFX



### JavaFX 개요

- 크로스 플랫폼에서 실행하는 리치 클라이언트 애플리케이션을 개발하기 위한 그래픽과 미디어 패키지를 말한다.
- 자바 7부터 JDK에 포함되어 있다.
- 패키지 연혁
  - AWT -> Swing -> JavaFX
- 어도비의 플래쉬(flash)와 마이크로소프트의 실버라이트(silverlight)의 대항마로 만들어졌다.
- javaFX 애플리케이션 구성 요소
  - 레이아웃
    - 자바 코드 파일 or FXML 파일
  - 외관 및 스타일
    - CSS 파일
  - 비즈니스 로직
    - 자바 코드 파일
  - 리소스
    - 그림 파일
    - 동영상 파일 등등



### JavaFX 애플리케이션 개발 시작

- 개발하려면 제일 먼저 메인 클래스를 작성해야한다.
